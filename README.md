### Technical Side:
| Language | Version |
| --- | --- |
| gcc (C Compiler) | 9.2.0 |
| java | 1.8.0 |
| Node.js | 14.16.0 |
| Python | 3.9.1 |

### Run Code
Example:
Compile *donut.c* with gcc
```sh
gcc donut.c
```
and run *.exe* file
<br/>
**or**
<br/>
Compile *Donut.java* with javac
```sh
javac Donut.java
```
and run *.class* file
<br/>
**or**
<br/>
Run for Node.js
```sh
node donut.js
```
**or**
<br/>
Run for Python
```sh
python donut.py
```
